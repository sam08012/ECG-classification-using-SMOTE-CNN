# ECG-classification-using-SMOTE-CNN
ECG classification using 10 ResNet CNN blocks, with 50 epochs. Uses SMOTE for oversampling of minority classes.
Results obtained:
## Precision: 97.0%
## Accuracy: 97.5% 
## Sensitivity: 97.0%
## F1 score: 0.9723
